This README file contains the instructions in order to run this project.

Access the project from the directory that it is saved on your local machine. 

Note that is must be from the terminal

Following this, enter within the folder and use the "node app.js" command in order to run the project locally.

The terminal will output the link "http://localhost:3000". Follow this link in order to view the application.

Doing this will bring the user the login page. Login with credientials and the wishlist app will be accessible.
